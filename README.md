# Native-Redirects
Create a native redirect link app.

This repository now includes a GitHub Pages-ready `index.html` that:
- Provides a mobile-friendly and desktop-friendly layout
- Includes a button for `tg://proxy?...` (opens Telegram app on supported devices)
- Includes a fallback `https://t.me/proxy?...` link
- Displays setup steps and the full proxy URL
